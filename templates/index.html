Sistema de Inventario

Este sistema fue desarrollado para apoyar la gestión de inventarios en MYPES del sector retail, donde frecuentemente se presentan errores en las operaciones de almacén, como registros incorrectos de productos, inconsistencias de stock y fallas en la actualización de inventarios. Para solucionar este problema, el sistema utiliza un procesador de lenguaje de dominio específico que interpreta, valida y transforma las operaciones de almacén mediante análisis léxico, sintáctico y semántico. De esta manera, permite automatizar procesos, reducir errores operativos y generar automáticamente instrucciones SQL Server para una gestión más eficiente y confiable del inventario.
